Questionnaire and Matching Process

Users can see which fraternities they best match with by taking our questionnaire, which lists five key attributes - Academics, Athletics, Leadership, Social, and Philanthropy. While these all hold some importance to fraternities, the level of importance varies by each organization. Each attribute has five buttons next to it, labeled one to five, with each number corresponding to how important each one is to the user. Specifically, the scale of importance for each attribute is as follows: 1 = not at all, 2 = very little, 3 = somewhat, 4 = very much, 5 = completely.

Our implementation allows users to easily update their values for each attribute if they made a mistake by clicking an incorrect number or simply want to update their choice of a value. When the user clicks the ‘Submit’ button, our algorithm calculates the match percentage between the user and each frat. After the algorithm has finished, the top three fraternities based on match percentage are displayed on the next page. This way, we create a more streamlined approach to help users determine which fraternities align most closely with their values.
